# Project Summary
CréeaStudio is a modern portfolio website designed for a multi-skilled professional specializing in communication management, graphic design, UX design, photography, and videography. The project aims to showcase the client's work through an elegant and interactive interface, enhancing user engagement and presenting a comprehensive view of their services.

# Project Module Description
- **Hero Section**: Engaging introduction with a creative background and call-to-action buttons.
- **About Section**: Information on the professional's skills and experience.
- **Services Section**: Detailed descriptions of services offered, including communication, graphic design, UX design, photography, and videography.
- **Portfolio Section**: A filterable grid showcasing various projects.
- **Testimonials Section**: Interactive carousel displaying client feedback.
- **Contact Section**: A form for potential clients to reach out directly.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Typed superset of JavaScript for better development experience.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool that provides a fast development environment.
- **PostCSS**: Tool for transforming CSS with JavaScript plugins.
- **Sonner**: Library for toast notifications.

# Usage
1. **Install Dependencies**: Run the command to install all necessary packages.
2. **Build the Application**: Use the build command to compile the application.
3. **Run the Application**: Start the development server to view the application locally.
